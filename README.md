
# 🚀 React + TypeScript + Vite 프로젝트

이 프로젝트는 **Vite** 기반의 **React + TypeScript** 웹 애플리케이션입니다.  
빠른 번들링, 타입 안정성, 개발 편의성을 갖춘 환경을 제공합니다.

<br/>

## 📦 기술 스택

- ⚛️ React 18
- 🌀 TypeScript
- ⚡ Vite
- 🧹 ESLint & Prettier (코드 품질 및 스타일 유지)
- 📦 Node.js (개발 환경)

<br/>

## 🛠️ 프로젝트 시작 방법

```bash
# 1. 의존성 설치
npm install

# 2. 개발 서버 실행
npm run dev

# 3. 빌드 (배포용)
npm run build
```

<br/>


📁 폴더 구조
```
.

├── .prettierrc          # Prettier 설정
├── index.html
├── tsconfig.json
├── vite.config.ts
└── README.md
```
